    BLW

    Main class used to configure, initialize and run blw library.

    About

    All Objects must extend this class or implement the \BLW\Interface\Object interface.

    Note:

    Whenever $DefaultOptions is used you must also define $_Initialized

    Object must implement dynamic properties

    since 1.0.0
    author mAsT3RpEE

    Adds an action.

    AddAction(string $Action) 
    Static

    About

    Actions are events that are triggered based on object events (ie Create).

    They are specific to a certain object id. An eample is AjaxElement object that uses actions to handle ajax requests.
